# Scrylixir

An Elixir wrapper for the [Scryfall API](https://scryfall.com/docs/api/).

### Status

![build](https://github.com/peterkrauz/scrylixir/actions/workflows/build.yml/badge.svg)

## Installation

Add the dependency to `mix.exs` such as:

```elixir
def deps do
  [
    ...
    {:scrylixir, "~> 0.1.0"}
  ]
end
```
